# SCA Inventory Simulation Workshop

This repository contains the teaching material, code, data preparation scripts, assignments, and assessment material for the SCA inventory simulation workshop.

The workshop teaches inventory simulation through a guided retail case. Inventory is the application context, but the core learning goal is simulation reasoning: dynamic state updates, stochastic input modelling, random number generation, Monte Carlo replications, output uncertainty, experiment design, and decision-making under risk.

## Repository structure

- `slides/`: lecture slides, exported PDFs, and slide figures
- `notebooks/`: teaching, student, and solution notebooks
- `data/`: raw, processed, and generated data files
- `scripts/`: reusable scripts for data generation, parameter calibration, plotting, and utilities
- `assignments/`: individual and group assignment prompts
- `quizzes/`: quiz briefs, question bank, assets, final versions, and answer explanations
- `docs/`: learning outcomes, evaluation criteria, and teaching notes
- `outputs/`: generated figures, notebooks, quizzes, and temporary outputs

## Teaching philosophy

The course should not only teach students to run code. It should teach them to interpret simulation evidence and make defensible operational decisions.

Good student work should show that they can:

1. diagnose demand and choose simulation inputs;
2. design a simulation experiment;
3. interpret stochastic outputs, uncertainty, and trade-offs;
4. make a recommendation aligned with a business decision profile.

## Notes for future development

This repository is intended to be managed with GitHub and Codex. Codex should help with implementation, consistency checks, notebook cleanup, generated data, and quiz assets. Pedagogical decisions should remain explicit and documented in the repository.
